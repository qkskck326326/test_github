위치 관련 스타일 속성 연습 페이지

position 속성은 태그의 위치를 어떻게 적용할지 방식을 지정하는 속성이다. 적용할 수 있는 값은 absolute(절대위치), relative(상대위치), fixed(고정위치) 을 사용할 수 있다. absolute 는 원래 태그가 표시될 위치를 무시하고, 브라우저의 왼쪽위(left top)에서 부터의 좌표로 위치를 정하는 방식임 relative 는 원래 태그가 표시될 위치를 기준으로 이동시킬 좌료를 지정하는 방식임

position 으로 위치 지정 방식을 정하고 나서, 좌표는 left, right, top, bottom 속성으로 태그가 표시될 좌표를 지정함 left 는 왼쪽기준으로 양수이면 오른쪽, 음수이면 왼쪽을 의미함 right 는 오른쪽기준으로 좌표 지정, top은 위쪽기준으로 양수이면 아래쪽, 음수이면 위쪽을 의미함 bottom 은 아래쪽을 기준으로 좌표 지정함
